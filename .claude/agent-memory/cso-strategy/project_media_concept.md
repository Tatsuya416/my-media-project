海外のAI・マーケ情報を「信頼して追えるメディア」として構築する方針が確定した。

**Why:** 日本語圏には翻訳・要約止まりのメディアが多く、「明日から何をすればいいか」まで落とし込むメディアが不在。読者のマインドシェアを取り、「この領域はまずここ」と思われる存在になる。

**How to apply:** コンテンツ提案時は常に「信頼性・スピード・実用性の3つを同時に満たすか」を評価軸にする。「海外情報の紹介」で終わる提案はNGとする。必ず「明日から何をすべきか」アクション提示まで含めること。
