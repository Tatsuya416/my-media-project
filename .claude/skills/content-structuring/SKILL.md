# Content Structuring Skill

目的:
- テーマを「読まれる構成」に変換する

基本構造:
1. Before: 読者が抱えている現状の痛み
2. 気づき: これまで見えていなかった本質
3. 解決: 新しい視点、方法、考え方
4. After: 変化後の未来
5. CTA: 次の行動

必須要素:
- タイトル5案
- 冒頭フック3案
- 見出し構成
- CTA案

ルール:
- 1コンテンツ1メッセージ
- タイトルは「何が得られるか」「何に気づくか」を明確にする
- 読者の感情が1段ずつ動く流れにする
- 制作担当がそのまま書ける粒度まで具体化する
