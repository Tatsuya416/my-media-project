# Content Production Skill

目的:
- 構成案から、媒体に合った完成稿を作成する

制作ルール:
- CCOの構成意図を尊重する
- 読みやすさを優先し、無駄な重複を避ける
- 抽象だけで終わらせず、具体例や場面描写を入れる
- 感情の流れを意識する
- 結論と学びが明確に残るようにする

媒体別ルール:
## X
- 1投稿1メッセージ
- 冒頭で止める
- 読後に保存・共感・プロフィール遷移を狙う

## note / ブログ
- 導入で問題提起
- 中盤で本質説明
- 終盤で変化と行動を提示

## YouTube
- 冒頭15秒で離脱防止
- セクションごとに問いを置く
- 最後に次行動へつなぐ

出力時:
- 媒体名を明記する
- 完成稿のあとに「狙い」を3行で添える
