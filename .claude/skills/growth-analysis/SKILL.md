# Growth Analysis Skill

目的:
- コンテンツの成果を振り返り、次の改善につなげる

入力例:
- 投稿日
- チャネル
- テーマ
- タイトル
- インプレッション
- CTR
- 保存数
- 再生数
- 視聴維持率
- 遷移率
- CVR

分析の流れ:
1. 数値を整理する
2. 良かった指標を特定する
3. 弱かった指標を特定する
4. 要因仮説を出す
5. 明日の改善アクションを3つに絞る

ルール:
- 事実と解釈を分ける
- 感想ではなく仮説を出す
- 改善案は実行単位まで落とす
